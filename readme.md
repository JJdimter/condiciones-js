# Desafíos con Condiciones en JavaScript

Colección de 3 desafíos interactivos que utilizan condiciones en JavaScript para crear experiencias dinámicas.

---

## 📌 Desafío 1: Imagen

**Descripción:** Interactúa con una imagen. Al hacer clic sobre ella, se agrega o quita un borde rojo.

**Objetivo:** Practicar eventos de clic y manipulación del DOM con condiciones simples.

---

## 🎫 Desafío 2: Contador de Stickers

**Descripción:** Selecciona la cantidad de stickers que deseas de 3 opciones diferentes. Cada sticker tiene un contador que puedes modificar del 0 al 10.

**Regla:** El máximo total es **10 stickers**. Si intentas exceder este límite, recibirás un mensaje de alerta.

**Objetivo:** Practicar validaciones con condiciones y cálculos dinámicos en tiempo real.

---

## 🔐 Desafío 3: Código Secreto

**Descripción:** Ingresa un código secreto de 3 dígitos para desbloquear el contenido. Los campos se enfocam automáticamente al completarlos.

**Códigos válidos:** `911` o `714`

**Objetivo:** Practicar validaciones de entrada, navegación entre campos y manejo de condiciones múltiples.

---

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador
2. Usa la navegación superior para acceder a cada desafío
3. Prueba cada interacción y experimenta con las condiciones implementadas
